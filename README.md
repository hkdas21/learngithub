# GitHub Certifications Study Guide

Comprehensive study materials for all five GitHub certification exams, hosted on GitHub Pages.

## Certifications Covered

| Code | Certification |
|------|--------------|
| GH-900 | GitHub Foundations |
| GH-100 | GitHub Administration |
| GH-200 | GitHub Actions |
| GH-300 | GitHub Copilot |
| GH-500 | GitHub Advanced Security |

## Each Certification Includes

- **Exam Overview** — Objectives, format, passing score, and preparation tips
- **Study Notes** — Module-by-module notes covering all exam domains
- **Labs** — Hands-on exercises with step-by-step instructions
- **Quiz** — Practice questions with expandable answers
- **Flashcards** — Key terms and concepts for quick review

## Local Development

```bash
# Install dependencies
bundle install

# Serve locally
bundle exec jekyll serve

# Visit http://localhost:4000/github-cert-study-guide/
```
