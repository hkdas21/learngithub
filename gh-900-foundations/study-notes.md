## Module 1: Introduction to Git and GitHub

### What is Git?

- Distributed version control system (DVCS)
- Tracks changes in source code during development
- Supports non-linear development through branching
- Every clone is a full backup of the repository

### What is GitHub?

- Cloud-based hosting service for Git repositories
- Adds collaboration features on top of Git
- Provides web UI, APIs, and CLI tools
- Offers free and paid tiers for individuals and organizations

### Key Concepts

- **Repository**: A project container with files and revision history
- **Branch**: A parallel version of the repository
- **Commit**: A snapshot of changes with a message
- **Clone**: A local copy of a remote repository
- **Fork**: A personal copy of someone else's repository
