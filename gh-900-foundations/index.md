## About This Certification

The **GitHub Foundations** certification validates your understanding of the foundational topics and concepts of collaborating, contributing, and working on GitHub.

## Exam Details

| Detail | Info |
|--------|------|
| **Exam Code** | GH-900 |
| **Duration** | 120 minutes |
| **Number of Questions** | 75 |
| **Passing Score** | 70% |
| **Format** | Multiple choice, multiple select |
| **Cost** | $99 USD |

## Domains Covered

| Domain | Weight |
|--------|--------|
| Introduction to Git and GitHub | 22% |
| Working with GitHub Repositories | 8% |
| Collaboration Features | 30% |
| Modern Development | 13% |
| Project Management | 7% |
| Privacy, Security, and Administration | 10% |
| Benefits of the GitHub Community | 10% |

## Preparation Tips

- Create a free GitHub account and practice daily
- Work through the GitHub Skills learning paths
- Focus heavily on collaboration features (30% of the exam)
- Understand the differences between Git and GitHub
- Review GitHub pricing tiers and feature availability

> **Tip:** This is the foundational cert — master this before moving to the specialized ones.
